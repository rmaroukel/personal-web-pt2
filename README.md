## How to Use Repository
**Clone Repo**
`git clone`

**Install dependencies**
`npm install`

**Initiating Tailwind CSS watch**
`npm run dev`

**Initiating Webpack build**
`npm run build`

**Initiating Webpack server**
`npm run start`
